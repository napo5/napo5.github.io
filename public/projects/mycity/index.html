MyCity, uno spazio per la gestione dei comuni

Feb 27, 2019 · 2 min read

La web app MyCity é il frutto di due esami di gruppo della mia triennale in Informatica, ovvero Ingegneria del Software e Programmazione Web. MyCity é un applicazione pensata per dispositivi sia desktop che mobile, realizzata in Java tramite i framework Spring Boot e Maven, gestendo le autenticazioni in cloud tramite Auth0.

Lo scopo di MyCity é creare un canale di comunicazione diretta tra i cittadini di un comune e l’amministrazione, permettendo ai cittadini di partecipare direttamente alla risoluzione di piccoli problemi sul terrorio. La progettazione dell’architettura é stata realizzata con il software Visual Paradigm.
